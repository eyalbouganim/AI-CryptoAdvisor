## üíª How to Run Locally

This project is configured to make local development easy and convenient, allowing both the frontend and backend to run simultaneously from the root directory.

### Prerequisites

You must have the following installed on your machine:
* **Node.js** (v18+)
* **npm** (comes with Node.js)
* **MongoDB** (mongoose)
* **MUI**

### 1. Installation

Navigate to the project root directory and install dependencies for both services.

### 2. Running The Servers

Open the first terminal and run:

```bash
cd backend
npm run dev
```

Then open a second terminal and run:

```bash
cd frontend
npm run dev
```

Both servers will be running and functional, and all is left for you to do is enjoy the web app!
